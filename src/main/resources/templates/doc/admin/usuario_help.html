Ayuda de usuario

En este listado podéis visualizar los datos personales del usuario y organizarlos por Id o apellidos.
Para crear un nuevo usuario, está indicado arriba a la izquierda el botón NUEVO. Las columnas que se ven son:

  • Id: Número de identificación.
  • Nombre.
  • Primer apellido.
  • Segundo apellido.
  • Teléfono.

En este formulario podéis visualizar las casillas en las que rellenar los datos personales del usuario.
Para que queden grabados, abajo a la izquierda de la pantalla está indicado el botón GUARDAR. Las casillas que se ven son:

  • Usuario: Identificación de usuario.
  • Password: Contraseña escogida por el usuario.
  • Nombre.
  • Primer apellido.
  • Segundo apellido.
  • Fecha de nacimiento.
  • Género: con el que se identifica el usuario.
  • DNI o documento de identificación.
  • Dirección.
  • Teléfono.
  • E-mail.
  • Localidad.
  • Nacionalidad.
  • Estado civil.
  • Número de hijos.
  • Nº de Seguridad Social.
  • Discapacidad: indicar el porcentaje en caso afirmativo.
  • Ingresos anuales: cantidad en bruto.
  • Condición personal: otra información adicional no comtemplada en las anteriores casillas.
  • Comentarios adicionales: otra información que crea relevante el usuario para completar su perfil de usuario.
